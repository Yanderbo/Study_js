# Study_js 快速指南

本仓库收集了 JavaScript 入门的课堂示例与学习笔记，方便在浏览器中直接练习和对照阅读。

## 目录概览
- `JavaScript入门.md`：按章节整理的入门笔记，包含基础语法、常用 API 以及示例代码。
- `class_one`、`class_two`、`class_three`：与笔记章节对应的 HTML 示例文件，可直接在浏览器中运行体验。
- `image`：笔记引用的配图资源。

## 试用方法
1. **直接打开**：克隆或下载本仓库后，双击 `class_*` 目录下的 HTML 文件，即可在浏览器中查看交互效果。
2. **通过本地服务器访问**：在仓库根目录运行 `python -m http.server 8000`，再在浏览器打开 `http://localhost:8000`，按目录进入对应示例。
3. **开发者工具调试**：在浏览器按 F12 或右键检查，随时修改脚本或样式并立即看到结果。

## 学习建议
- 建议先阅读 `JavaScript入门.md` 了解概念，再结合对应的 HTML 示例边学边改。
- 示例涵盖基础弹窗、字符串处理、模板字符串等操作；可在控制台尝试输入变量、修改 DOM，观察每一步的效果。
- 如果需要保存自己的尝试，可以复制示例文件后另存并重命名，不会影响原始范例。
